# Travel Landing Page

A responsive landing page for a fictional travel agency.  
It features a typing effect in the header, a modal with a special offer, a section showcasing popular destinations, and a footer with contact information, quick links, social media, and a newsletter subscription form.

## Technologies Used
- HTML5 for semantic structure
- CSS3 for responsive design, layout, and animations
- JavaScript (ES6) for interactivity and dynamic behavior

## Project Structure
- `index.html` – Main page
- `style.css` – Styling and layout
- `script.js` – Interactive functionality

## Features
- Typing effect in the header
- Modal popup with exclusive travel deals
- Scroll-based screen transitions
- Destination cards with hover effects
- Newsletter subscription with local storage support
- Responsive footer with links, contact details, and testimonials

## Installation and Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/travel-landing-page.git
